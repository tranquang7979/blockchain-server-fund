# DEPLOY A CONTRACT
1. open contract scripts at https://remix.ethereum.org/
2. left menu, click Deploy & run transaction
    + Environment = Inject Web3
    + click button Deploy
3. for review => go to https://rinkeby.etherscan.io/tx/[copy and paste address of contract]    
4. verify and publish
    + go to https://rinkeby.etherscan.io/address/[copy and paste address of contract]
    + click on tab Contract
    + click Verify and Publish and do step 5.
5. key in contract information for verify
    + Compiler Type = Solidity (Single file)
    + Compiler Version = v0.8.4+commit.c7e474f2
    + Open Source License Type = 3) MIT License (MIT)
    + click Continue button
    + copy source code of contract to the text field "Solidity Contract Code".
    + click Verify & Publish 
    + after few minutes, you will see message "Successfully generated ByteCode and ABI for Contract Address..."       

# USING CONTRACT
1. copy ABI
    + left menu => click Solidity Compiler
    + at the bottom, click the copy icon (ABI)
